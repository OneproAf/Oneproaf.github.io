InstaText AI

Smart Captions Generator for Social Media

📌 Project Description

InstaText AI is an AI-powered tool that generates unique captions for your photos on Instagram, Facebook, and X. Just upload a photo, choose a style — and get ready-made captions with hashtags.

🛠️ Features

  • 📷 Upload photos from mobile or desktop
  • 🎭 Choose caption style: Funny, Romantic, Smart, Deep, Trendy
  • 🧠 Generate 3 caption options using GPT-4
  • 🔗 Auto-generated relevant hashtags
  • 📋 One-click copy text
  • 📱 Fully mobile responsive
  • 🌐 Hosted on GitHub Pages, no backend needed
